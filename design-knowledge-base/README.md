# Design Knowledge Base

This directory contains design assets, tokens, and guidelines imported from BuildLoop.

## Structure

- **`branding/`** - Logo files and brand assets (SVG logos, color schemes, button styles)
- **`design-tokens/`** - Aura design tokens JSON file for UI theming
- **`design-guidelines/`** - Design knowledge graph with patterns, guidelines, and documentation

## Contents

### Branding Assets
- `dejoule-logo.svg` - Primary DeJoule logo
- `dejoule-logo-dark.svg` - Dark theme variant
- `dejoule-logo-white.svg` - White/background variant
- `Smart-joules-logo-red.svg` - Smart Joules logo (red)
- `colors.png` - Color palette reference
- `buttons.png` - Button design reference

### Design Tokens
- `tokens-may11.json` - Comprehensive Aura design token set with 3390+ tokens for:
  - Colors (primitive, semantic, component)
  - Spacing and sizing
  - Typography (font sizes, weights, line heights)
  - Borders, shadows, and radius
  - Light and dark theme variants

### Design Guidelines
The `design-guidelines/` folder contains:
- Knowledge graph structure for design patterns
- Icon library documentation (Boxicons)
- Figma file mappings
- Design tokens documentation
- UI element guidelines
- Mockup-to-requirements patterns

## Source

Imported from [SmartJoules/BuildLoop](https://github.com/SmartJoules/BuildLoop)

## Usage

These assets can be referenced by:
- AI agents for design consistency
- Developers for UI theming and component styling
- Designers for brand guidelines and patterns
- QA for visual regression testing baselines
